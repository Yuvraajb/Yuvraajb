## Projects & Builds

**Mission Artemis** · *Apple Swift Student Challenge 2026 Winner*
Immersive educational iOS app simulating the Artemis II crewed lunar flyby mission. Cinematic onboarding (Neil Armstrong audio, hold-to-ignite), SceneKit 3D orbital views, Swift Charts telemetry, Core Haptics, engineering challenges at each mission phase, on-device crew chat via Foundation Models.
`SwiftUI` `SceneKit` `Foundation Models` `Swift Charts` `Core Haptics` `SwiftData`

**CRIS.ai** · *ElevenLabs Track Winner — Hack4Humanity*
Real-time AI crisis response system. Streaming STT → Groq LLM inference → ElevenLabs neural TTS. Multi-turn context management for stateful, low-latency voice interaction.
`ElevenLabs` `Groq` `Streaming STT` `LLMs` `Voice Pipelines`

**Homeability.ai** · *Hackathon Build — AWS Track*
LLM-powered housing accessibility analyzer. Unstructured user input → personalized recommendations via AWS Bedrock. Serverless backend on Lambda for low-latency concurrent inference.
`AWS Bedrock` `AWS Lambda` `Node.js` `LLM Pipelines`

**SCIF × AI Club @ SCU** · *Active*
Building applied AI tooling at the intersection of SCU's Science & Innovation Forum and the AI Club — research-focused ML project for the university community.
`Python` `Applied ML` `Research`

**Sports Business Club Website** · Full-stack from scratch. Dynamic events, member dashboards, admin CMS, REST APIs for live sports data. Cut content overhead 60%.
`ReactJS` `Node.js` `REST APIs`

**UMA Website @ SCU** · NextJS + TailwindCSS production site. Built AI-powered event highlights summarizer (OpenAI API) — cut post-event turnaround 70%.
`NextJS` `TailwindCSS` `OpenAI API`
